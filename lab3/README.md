## Опис роботи

Лабораторна робота присвячена візуалізації багатовимірних даних, розвідувальному аналізу (EDA) та пошуку прихованих закономірностей з використанням бібліотек Matplotlib та Seaborn. Лабораторна робота базується на класичному наборі даних Heart Disease (UCI Machine Learning Repository).

Робота включає наступні етапи:
* **Завантаження та очищення даних:** Автоматизоване отримання даних, обробка пропущених значень (NaN) у колонках `ca` та `thal` методом заповнення медіаною, перетворення типів даних для подальшого аналізу.
* **Базова візуалізація:** Побудова точкових графіків (Scatter Plot) із лініями тренду для дослідження залежності максимальної частоти серцевих скорочень від віку пацієнта, а також гістограми з 5 ціновими діапазонами рівня холестерину.
* **Багатовимірна візуалізація:** Створення складних графіків для аналізу понад двох змінних одночасно, зокрема матриці попарних залежностей (Pair Plot), графіка паралельних координат (Parallel Coordinates) та Bubble Chart.
* **Статистичні графіки:** Дослідження розподілу ЧСС та холестерину за статтю і діагнозом (Violin Plot, Box Plot), структури пацієнтів за рівнем ЧСС (Donut Chart), а також побудова теплової карти (Heatmap) для аналізу кореляції між числовими атрибутами датасету.

## Вимоги до системи

* **Версія Python:** 3.10 або вище
* **Пакетний менеджер:** pip

## Інструкція із запуску

1. Відкрийте термінал у кореневій директорії проєкту (наприклад, `lab3`).

2. Створіть віртуальне середовище:
   ```bash
   python -m venv venv
   ```

3. Активуйте віртуальне середовище:
   * **Windows:** `venv\Scripts\activate`
   * **Linux / macOS:** `source venv/bin/activate`

4. Встановіть необхідні залежності з файлу `requirements.txt`:
   ```bash
   pip install -r requirements.txt
   ```

5. Запустіть середовище Jupyter Notebook:
   ```bash
   jupyter notebook
   ```
