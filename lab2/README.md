## Вимоги до системи

- Python 3.10+
- pip

## Встановлення залежностей

Створити віртуальне середовище:

```
python -m venv venv
```

Активувати віртуальне середовище:

- Windows:
```
venv\Scripts\activate
```

- Linux/Mac:
```
source venv/bin/activate
```

Встановити залежності:

```
pip install -r requirements.txt
```
