# Lab 2 – Наука про дані: підготовчий етап

## Опис завдання

Лабораторна робота присвячена підготовці та аналізу даних з використанням Python, pandas та Jupyter Notebook.

Частина 1:
- створити віртуальне середовище (venv);
- завантажити файли зі значеннями VHI-індексу для адміністративних одиниць України;
- зчитати завантажені файли у pandas DataFrame;
- виконати очищення даних (data cleaning);
- змінити індексацію областей відповідно до української абетки;
- реалізувати функції для формування вибірок:
  - ряд VHI для області за вказаний рік;
  - ряд VHI за діапазон років для вибраних областей;
  - пошук мінімуму, максимуму, середнього значення та медіани.

Частина 2:
- завантажити датасет Individual Household Electric Power Consumption Dataset;
- виконати очищення даних;
- сформувати вибірки з використанням pandas;
- проаналізувати час виконання процедур за допомогою модуля timeit;
- виконати нормалізацію та стандартизацію даних;
- обчислити коефіцієнти кореляції Пірсона та Спірмена;
- виконати One Hot Encoding категоріального атрибута.

Обидві частини реалізовані у вигляді окремих Jupyter Notebook файлів.

## Інструкція із запуску

1. Встановити Python (версія 3.9 або новіша).

2. Створити віртуальне середовище:

python -m venv venv

3. Активувати середовище:

Windows:
venv\Scripts\activate

Linux / MacOS:
source venv/bin/activate

4. Встановити необхідні бібліотеки:

pip install pandas numpy jupyter

5. Запустити Jupyter Notebook:

jupyter notebook

6. Відкрити та виконати файли:

lab2_part1.ipynb  
lab2_part2.ipynb
