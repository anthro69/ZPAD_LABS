# Лабораторна робота №4 — Візуалізація даних 2

## Опис

Інтерактивна програма для візуалізації гармонічного сигналу виду:

$$y(t) = A \cdot \sin(\omega t + \varphi)$$

з накладеним гаусівським шумом та фільтрацією за допомогою фільтра Батерворта (low-pass).

## Функціонал

- Відображення **чистої гармоніки** та **зашумленого сигналу** на першому графіку
- Відображення **відфільтрованого сигналу** поряд із зашумленим та ідеальним на другому графіку
- Інтерактивні **слайдери** для керування параметрами:
  - `Amplitude` — амплітуда гармоніки
  - `Frequency` — частота гармоніки
  - `Phase` — фазовий зсув
  - `Noise Mean` — середнє значення шуму
  - `Noise Cov` — дисперсія шуму
  - `Cutoff Freq` — гранична частота фільтра
- **Чекбокс** `Show Noise` — вмикає/вимикає відображення шуму
- **Кнопка** `Reset` — відновлює початкові параметри

## Особливості реалізації

- Зміна параметрів гармоніки **не перегенеровує шум** — шум залишається тим самим
- Зміна параметрів шуму **не змінює гармоніку** — перегенеровується лише шум
- Фільтр: `scipy.signal.butter` (Butterworth low-pass) + `filtfilt` (нульовий фазовий зсув)

## Інструкція користувача

| Елемент | Дія |
|---|---|
| Слайдер `Amplitude` | Змінює амплітуду гармоніки |
| Слайдер `Frequency` | Змінює частоту гармоніки |
| Слайдер `Phase` | Змінює фазовий зсув |
| Слайдер `Noise Mean` | Змінює середнє шуму (перегенеровує шум) |
| Слайдер `Noise Cov` | Змінює дисперсію шуму (перегенеровує шум) |
| Слайдер `Cutoff Freq` | Змінює граничну частоту фільтра |
| Чекбокс `Show Noise` | Показати або сховати шум на графіках |
| Кнопка `Reset` | Відновити всі параметри до початкових значень |
