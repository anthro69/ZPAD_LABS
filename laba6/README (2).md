## Про програму

Програма реалізує захоплення відеопотоку з веб-камери та обробку кадрів у реальному часі засобами бібліотеки OpenCV на Linux. Код організований за модульним принципом — кожен клас розміщено в окремих `.hpp` та `.cpp` файлах.

## Режими роботи та керування

Програма підтримує кілька режимів обробки зображення, які перемикаються з клавіатури:

**Клавіші:**
* `0` — без обробки (оригінальний кадр)
* `1` — негатив (інверсія кольорів)
* `2` — розмиття за Гаусом
* `3` — виділення країв (фільтр Canny)
* `4` — градієнтний фільтр Собеля
* `5` — порогова бінаризація
* `Q` / `ESC` — вихід з програми

**Мишка:**
* Прокрутка колесом — плавний зум від 0.2x до 5.0x

**Елементи інтерфейсу:**
* Трекбар **Brightness** — змінює яскравість відеопотоку в діапазоні 0–200
* На кадрі відображається поточний FPS, номер кадру та активний масштаб

## Системні вимоги

* **ОС:** Linux (перевірено на Kali Linux)
* **Залізо:** процесор від 1.5 GHz, RAM від 1 GB, підключена веб-камера (`/dev/video0`)
* **Для збірки:** GCC з підтримкою C++17, CMake 3.10+, Make, OpenCV 4.x, libv4l-dev

## Збірка та запуск

Клонуємо репозиторій:
```bash
git clone <url>
cd lab6_opencv
```

Встановлюємо залежності:
```bash
./preinstall.sh
```

Компілюємо:
```bash
./build.sh
```

Запускаємо:
```bash
./run.sh
# або з іншим індексом камери:
./run.sh 1
```

## Структура проєкту

```
lab6/
├── CMakeLists.txt
├── preinstall.sh
├── build.sh
├── run.sh
├── README.md
├── requirements.txt
├── include/
│   ├── CameraProvider.hpp
│   ├── KeyProcessor.hpp
│   ├── FrameProcessor.hpp
│   └── Display.hpp
└── src/
    ├── main.cpp
    ├── CameraProvider.cpp
    ├── KeyProcessor.cpp
    ├── FrameProcessor.cpp
    └── Display.cpp
```

| Клас | Призначення |
|---|---|
| `CameraProvider` | Читання кадрів з камери через `cv::VideoCapture` |
| `KeyProcessor` | Обробка натискань клавіш, зберігання режиму (`enum Mode`) |
| `FrameProcessor` | Застосування фільтрів, зуму, яскравості та HUD |
| `Display` | Виведення готового кадру у вікно (`cv::imshow`) |
