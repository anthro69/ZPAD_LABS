## Опис роботи

Ця програма демонструє роботу з потоковим відео з веб-камери за допомогою бібліотеки OpenCV у середовищі Linux. Проєкт має модульну архітектуру (класи винесені в окремі файли) та підтримує інтерактивну обробку кадрів у реальному часі.

## Функціонал та керування

Під час роботи програми можна перемикати режими обробки зображень та взаємодіяти з відеорядом:

**Клавіатура (режими обробки):**
* `0` — Звичайний режим (Normal)
* `1` — Інверсія кольорів (Invert)
* `2` — Розмиття Гауса (Gaussian Blur)
* `3` — Фільтр Canny (виділення контурів)
* `4` — Фільтр Собеля (градієнти)
* `5` — Бінаризація (Binary threshold)
* `Q` або `ESC` — Завершення роботи програми

**Миша:**
* Колесо прокрутки — збільшення / зменшення зуму (0.2x – 5.0x)

**Інтерактивні елементи:**
* Слайдер **Brightness** — регулювання яскравості кадру (0–200, нейтраль = 100)
* HUD на екрані — відображення поточного FPS, лічильника кадрів та коефіцієнта зуму

## Вимоги до операційної системи та заліза

* **Операційна система:** Linux (протестовано на Kali Linux)
* **Апаратне забезпечення:**
  * Процесор: 1.5 GHz або швидший
  * Оперативна пам'ять: від 1 GB
  * Наявність підключеної веб-камери (пристрій `/dev/video0` або аналогічний)
* **Залежності для збірки:**
  * Компілятор з підтримкою C++17 (GCC/G++)
  * CMake версії 3.10+
  * Make
  * Бібліотека OpenCV версії 4.x
  * libv4l-dev

## Інструкція для збірки та запуску

Клонуйте репозиторій та перейдіть у робочу директорію:
```bash
git clone <url репозиторію>
cd lab6_opencv
```

Надайте права на виконання скриптів (за потреби):
```bash
chmod +x preinstall.sh build.sh run.sh
```

Встановіть залежності (потребує прав адміністратора):
```bash
./preinstall.sh
```

Зберіть проєкт:
```bash
./build.sh
```

Запустіть програму:
```bash
./run.sh
```

За потреби вкажіть індекс камери як аргумент (за замовчуванням `0`):
```bash
./run.sh 1
```

## Архітектура проєкту

```

├── CMakeLists.txt
├── preinstall.sh

├── run.sh
├── README.md
├── requirements.txt
├── include/
│   ├── CameraProvider.hpp
│   ├── KeyProcessor.hpp
│   ├── FrameProcessor.hpp
│   └── Display.hpp
├── src/
│   ├── main.cpp
│   ├── CameraProvider.cpp
│   ├── KeyProcessor.cpp
│   ├── FrameProcessor.cpp
│   └── Display.cpp
└── build/
```

| Клас | Відповідальність |
|---|---|
| `CameraProvider` | Захоплення кадрів з камери (`cv::VideoCapture`) |
| `KeyProcessor` | Обробка клавіш, зберігання поточного режиму (`enum Mode`) |
| `FrameProcessor` | Обробка зображень: фільтри, зум, яскравість, HUD |
| `Display` | Відображення результату (`cv::imshow`) |
