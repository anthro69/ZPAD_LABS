## Про проєкт

Веб-додаток для інтерактивного аналізу вегетаційних індексів (VCI, TCI, VHI) по регіонах України. Дані автоматично завантажуються з відкритого сервера NOAA при першому запуску — нічого не потрібно завантажувати вручну.

## Вимоги до системи

Версія Python: 3.10 або вище
---
Інструкція із запуску
---
## Створіть віртуальне середовище:
```bash
python -m venv venv
```
## Активуйте віртуальне середовище:

Windows: venv\Scripts\activate
Linux / macOS: source venv/bin/activate

## Як запустити

```bash
pip install -r requirements.txt
streamlit run laba5.py
```

Додаток відкриється у браузері за адресою `http://localhost:8501`.  
При першому запуску автоматично завантажуються дані (~1-2 хв), далі використовується локальний кеш.

---

## Що вміє додаток

- Вибір індексу (VCI / TCI / VHI) та регіону через dropdown
- Фільтрація за діапазоном тижнів і років через слайдери
- Сортування таблиці за зростанням або спаданням
- Скидання всіх фільтрів до початкового стану
- Три вкладки: таблиця / графік / порівняння всіх областей
- Графік часового ряду з підсвіткою зон посухи
- Bar chart медіан та box plot по всіх регіонах
